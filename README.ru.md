# VPN Proxy Manager

[🇬🇧 English version](README.md)

Плагин для [exteraGram](https://exteragram.app) и [AyuGram](https://ayugram.one), который автоматически отключает прокси при активном VPN на устройстве и восстанавливает его после отключения VPN.

---

## Как работает

Плагин запускает фоновый мониторинг, который каждые 3 секунды проверяет наличие активного VPN через Android `ConnectivityManager`. При изменении состояния:

| Событие | Действие |
|---|---|
| VPN **включён** | Прокси **отключается** |
| VPN **выключен** | Прокси **восстанавливается** (только если был отключён этим плагином) |

---

## Возможности

- 🔄 Автоматическое управление прокси в зависимости от состояния VPN
- 🔔 Всплывающие уведомления при изменении состояния прокси
- ⚙️ Переключение авто-управления прямо из настроек плагина
- 🛡 Безопасное восстановление — прокси включается обратно только если плагин сам его отключил

---

## Установка

1. Скачай последний `.plugin` файл из [Releases](../../releases)
2. Открой exteraGram или AyuGram
3. Перейди в **Настройки → Плагины → Установить из файла**
4. Выбери скачанный файл и включи плагин

---

## Требования

- exteraGram `11.12.0`+ или AyuGram (только полная версия — Lite не поддерживает плагины)
- В настройках Telegram должен быть заранее настроен прокси

---

## Настройки

| Настройка | Описание | По умолчанию |
|---|---|---|
| Автоуправление прокси | Включить/выключить основную функцию | ✅ Вкл |
| Показывать уведомления | Всплывающее уведомление при изменении прокси | ✅ Вкл |
| Кнопка в боковом меню | Быстрое переключение из бокового меню | ✅ Вкл |
